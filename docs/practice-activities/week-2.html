Writing Functions

  1. Write a function called summ_stats() that takes a vector x as input and produces the following output:
  • for numeric variables: returns the mean, median, standard deviation, and IQR as a data frame
  • for categorical variables: returns the number of levels (categories) of the variable

Hint: You can use tibble() to create the data frame. For example, tibble(a = 1:2, b = 2:3) creates a data frame with variables a and b.

Iterating Functions

  1. Use map() to apply your summ_stats() function to every column in the diamonds dataset. Hint: Look up the bind_rows() documentation from dplyr to combine summary statistics for all quantitative variables into one data frame. The .id argument will be especially helpful in adding the variable names!
